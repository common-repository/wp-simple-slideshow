You can add a slide show very easily.

== Description ==

Upload the image in the config page, just write a short code.
Simple to see the slideshow.
I corresponds to the trimming of the uploaded image.

管理画面から画像をアップロードし、ショートコードを書くだけでスライドショーが表示できます。
アップロードした画像のトリミングに対応します。

== Installation ==

1. Upload `wp_simple_slideshow` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload your favorite photos in the config page.
4. Write [wp_simple_slideshow] short code where you want to see the slideshow.
